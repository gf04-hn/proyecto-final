Este Repositorio contiene las aplicaciones solicitadas como parte del proyecto final de la Clase "Introducción a Python" impartido por Tecno 3F.

Las aplicaciones que optamos por desarrollar son las siguientes:

GENERADOR DE TICKETS: Su función consta de generar tickets de soporte técnico, brindado numeración aleatoria (como lo indica la consigna) y almacenando cada uno en un archivo txt para poder consultarlos cuando sea necesario.

GENERADOR DE CONTRASEÑAS: Su propósito es generar contraseñas que se almacenan en un archivo txt, las cuales posteriormente podrán ser utilizadas en las correspondientes aplicaciones para las que fueron creadas. El sistema permite decidir el tipo de caracteres que se desea que contengan dichas contraseñas y permite que éstas sean generadas de forma manual o aleatoria, de igual forma el código evita que si se generan de manera aleatoria, se dupliquen con las previamente almacenadas.
